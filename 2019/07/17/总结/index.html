总结

Posted on 2019-07-17

第一部分

上午主要学习了使用hexo搭建博客并部署到GitHub上面
其中遇到了一些问题，下面是总结

1

首先是遇到了 git不是内部或外部命令的问题，这是由于未安装git工具，于是去网上下载，因为官网下载速度太慢，找到了腾讯软件中心进行下载，下载速度能到几M/s，之后遇到类似问题也可以如此解决。

2

其次是在部署到github上的时候，就是输入 hexo -d 的时候 显示错误：please tell me who your are
可以输入下面两行命令
git config –global user.email “邮箱”
git config –global user.name “名字”
告诉你的GitHub注册时的邮箱 以及用户名

第二部分

下午主要是继续学习了之前的springMVC课程，今天主要是复习了jsp的九大内置对象，学习了文件上传和下载
好像没有遇到什么问题，之后遇到问题就先记下来，不然容易忘记。

晚上继续学了一会然后看了一会Markdown的语法

hexo写日志的步骤如下

先在blog目录下，就是建立博客的目录下创建一个新的md文件
hexo new $fileName
然后进入/source/_post目录
然后使用vim编辑器进行编辑。
然后在本地观看效果
hexo server

最后更新到GitHub上
hexo d
